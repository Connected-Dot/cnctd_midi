# cnctd_midi

MIDI device interaction library for Rust.

## Installation

```toml
[dependencies]
cnctd_midi = { git = "https://github.com/Connected-Dot/cnctd_crates.git" }
```

## Features

- MIDI input handling
- Controller mapping
- Device enumeration
